Check EA-Utils¶

Check if the command exist in the environment

[3]:

which fastq-mcf
which fastq-multx

/usr/bin/fastq-multx

Check the version

[4]:

Usage: fastq-mcf [options] <adapters.fa> <reads.fq> [mates1.fq ...]
Version: 1.04.807

[5]:

Version: 1.02.772
