Trim the adapters in pair-end reads using Fastq-mcf¶

[4]:

fastq-mcf $MYINFO/TruSeq_adapter.fasta \
    $RAW_FASTQS/SRR4841864_1.fastq \
    $RAW_FASTQS/SRR4841864_2.fastq \
    -q 20 -x 0.5 \
    -o $TRIMMED/SRR4841864_1.trim.fastq \
    -o $TRIMMED/SRR4841864_2.trim.fastq
